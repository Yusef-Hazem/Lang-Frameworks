# Lang-Frameworks

A collection of practical implementations built with the LangChain ecosystem, exploring Retrieval-Augmented Generation (RAG), agentic workflows, retrieval strategies, and graph-based orchestration using **LangChain** and **LangGraph**.

## Tech Stack

* Python
* LangChain
* LangGraph
* FAISS
* Hugging Face Embeddings
* Groq
* Llama 3.3 70B
* FlashRank
* ColBERT

## Current Implementations

| Notebook                     | Description                                                                       |
| ---------------------------- | --------------------------------------------------------------------------------- |
| **01_FirstRag**              | End-to-end RAG pipeline with chunking, embeddings, and vector search              |
| **02_QueryTransformations**  | Multi-Query, RAG Fusion, HyDE, Step-Back Prompting, and query decomposition       |
| **03_Routing**               | Logical and semantic routing for retrieval and prompt selection                   |
| **04_Indexing_to_VectorDBs** | Multi-vector retrieval, multi-representation indexing, and ColBERT                |
| **05_Retrieval_Mechanisms**  | Re-ranking, ensemble retrieval, long-context reordering, and self-query retrieval |
| **06_Self_Reflection_RAG**   | Self-reflective RAG workflow with retrieval grading and hallucination detection   |
| **07_Adaptive_RAG_Agent**    | Adaptive RAG agent with routing, web search, and self-correcting generation       |
